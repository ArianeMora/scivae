Installing sci-vae¶

Note: It is strongly recommended to create a virtual environment for each version of sci-vae installed.

Python using pip¶

pip install scivae

R¶

I will add it as an R package soon, but until then check out the R example using Reticulate!

Note: if you are using R you need to make sure you have python installed. We strongly recommend using anaconda and creating an envrironment and then installing scivae in a particular envrionment using reticulate.

Checkout the examples in R for more details.
